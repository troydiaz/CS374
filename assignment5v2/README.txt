*   Compile the program as follows:
*
*       make clean && make all
*  
*

*
*       ./enc_server <port number1> &

*
*   Run the grading script to test all functionality:
*
*       ./p5testscript <port number1> <port number2> > graderesults 2>&1
*
*   View the test results in the file `mytestresults`:
*
*       cat graderesults
